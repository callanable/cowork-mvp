Cowork · Skills Marketplace

Install once. Every employee gets it.

A marketplace of installable, governable AI skills. Skills compose the MCPs your org already has — calendar, email, drive, CRM — into one-command workflows your team uses without thinking about it.

This is the MVP marketplace — one plugin today, built to demonstrate the pattern. The install primitive is Claude Code's /plugin marketplace; the unit of capability is a skill; the governance layer is whatever MCPs you've scoped to that user. The platform is mostly composition.

See the repo for the design doc on how this generalizes — memory layer, RBAC via MCP scoping, PR-based shipping.
